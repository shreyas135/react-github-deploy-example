# react-github-deploy-example
How to deploy React Project On Github Page?

# 1). Create Repository on Git (Ex. react-github-deploy-example)
# 2). git clone https://github.com/shreyas135/react-github-deploy-example.git
# 3). add "homepage" : "http://shreyas135.github.io/react-github-deploy-example" in package.json
# 4). git init
# 5). git remote add origin git@github.com:shreyas135/react-github-deploy-example.git
# 6). git add .
# 7). git commit -m "init"
# 8). git push origin main
# 9). yarn run deploy
